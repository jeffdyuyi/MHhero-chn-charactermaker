# Find Skills

This skill helps you discover and list all available skills in the workspace.

## What It Does

- Scans the `.trae/skills/` directory for available skills
- Lists all skills with their names and descriptions
- Provides information about each skill's purpose and usage

## When to Use

Invoke this skill when:
- You want to see what skills are available
- You need to find a specific skill
- You're curious about what skills can help with your tasks

## Usage Example

When you ask "What skills are available?" or "List all skills", this skill will provide a comprehensive list of all skills in the workspace.
