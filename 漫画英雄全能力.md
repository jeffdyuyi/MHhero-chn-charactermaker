### 🧬 变形系 (Alteration) 特殊能力资料库

| 特殊能力名称 | 详细描述 (完全依照原文) |
| :--- | :--- |
| **能力增幅** | 你可以选择一项能力(属性或特殊能力)提升到与能力增幅相同的等级，持续的画页数与能力等级相同。一旦该项能力“恢复”正常，即会在同等数目的画页之内降低一个等级。如果用于增幅的能力原本就是0级(比如本身并不拥有该项能力)，则会在恢复正常后降低到0级，在规定时间内无法使用。 |
| **能力提升** | 你可以将一项能力(属性或特殊能力)永久提升到6级或永久提高2级(最高不超过10级)，两者取较大值。该项特殊能力不计入决意等级，但如果提升后的属性达到7级(或更高)，则需要扣除1个决意点。 |
| **释放自我** | 你可以转变成为另外一个截然不同的角色，创造第二个自我！新角色自动失去一项特殊能力(用于生成新人格)。为了保持两个人格的一致性，游戏主持人可以要求新角色保留某些能力，尤其是保留能力来源和精神能力，但这样做并不是必要的。如果你的英雄只是一个身无特殊能力的“普通”人类，变成另外一个角色就会成为一项挑战(偶尔会变成麻烦)，而不是释放自我能力的应用。 |
| **变化形态** | 经过一个画页的预先准备，你可以转变成为血肉之躯以外的形态，从其他形态恢复到正常形态无须预先准备，可以在自己的画格内立即生效。如需增加附带效果，可以去掉变化形态的准备时间。变化形态状态下，角色拥有10级维系生命能力(请参阅下文的维系生命)。<br><br>● **能量形态**：你可以变成连贯的能量场(具体范例请参阅能量控制)，获得与变化形态同等级别的飞行能力，能够免疫物理伤害，但会失去力量属性，无法触及实体目标，也不能对它们造成影响。能对该类型能量生效的攻击同样也会对你造成影响。如需增加附带效果，可以选择同类能量进行能量控制。<br>● **爆发形态**：你可以使身体像附带爆溢效果的冲击波一样产生爆炸，在下一画页开始时即可恢复如常，在此期间能够免疫物理伤害，但无法对物质世界造成影响。你可以推迟恢复原状的时间，延后的画格数不能超过变化形态的能力等级，复原地点也可以在爆炸所及的远程范围内任意选择。如需增加附带效果，可以选择与变化形态同等级别的瞬间移动能力，自爆后在更远的范围内恢复原状。<br>● **流体形态**：你可以变成流体形态从裂缝和其他狭小空间之中流淌而过。你可能呈现液体形态(比如水)，也可能呈现微粒聚集形态(比如灰尘或沙土)，同时获得与变化形态同等级别的伤害抵抗能力和伸缩自如能力。<br>● **气体形态**：你可以变成一团气体、细小的悬浮粒子、甚至一群飞舞的昆虫或纳米机器人，获得1级飞行能力，能够从所有非密封开放空间之中穿行而过，同时免疫物理伤害，除非该次攻击能够破坏云状物质，或对它们造成影响。<br>● **阴影形态**：你可以变成与正常外观对应的扁平剪影，失去力量属性，无法触及实体目标，也不能对它们造成影响，同时能够免疫物理攻击，但是与光有关的攻击会对你造成额外伤害(将普通成功视为重大成功)。处于阴影或昏暗灯光之中时，你可以获得与变化形态同等级别的墙面爬行能力和隐身能力。<br>● **固体形态**：你可以变成金属/石头等高密度的固体物质，获得与变化形态同等级别的力量值，或是正常力量值+1，两者取较大值，同时获得与变化形态同等级别的伤害抵抗能力。 |
| **水下生存** | 拥有水下生存能力的角色既能在水中活动，又能在陆地活动，并且可以在水下呼吸，此时你的协调和感知等级等同于水下生存的能力等级，或者等同于正常属性等级+1，两者取较大值。根据水下生存能力等级的一半(向上取整)对照基准表，即可得到你的游泳速度。如需增加附带效果，可以提高角色在水下时的勇猛和力量等级，使其与水下生存的级别相同，或者提高1个等级，两者取较大值。 |
| **浓缩密度** | 你可以随意提高身体的密度，此时你的力量等级等同于浓缩密度的能力等级，或者等同于正常力量等级+1，两者取较大值，并获得与浓缩密度同等级别的伤害抵抗能力。如需增加限制条件，可以让你的协调等级等同于正常协调等级，或者等同于(10-浓缩密度的能力等级)，两者取较小值(不低于1级)。如需增加附带效果，可以通过降低身体密度脱离固定形态来获得穿越固体能力。 |
| **基因复制** | 可以制造你的复制体，需要一个画页的预先准备，其数量不能超过基因复制的能力等级，1级能力可以制造一个复制体，2级能力可以制造两个复制体，以此类推。复制体与你能力相同(没有基因复制这项能力，无法继续复制)，行动一致(适用于齐心协力规则，请参阅基本概念章节的齐心协力部分)，没有决意等级，但可以共用你的决意点数。如果死亡或失去意识，复制体将消散不见，如果使用者丧失行动能力，所有复制体都将不复存在。 |
| **三头六臂** | 从你的肩部、背部、身体两侧、甚至头部(或许是毛发状的触手卷须)长出强壮有力的触须。<br><br>● **外壳**：身体表面覆盖一层硬壳，获得与三头六臂同等级别的伤害抵抗能力。<br>● **利爪**：获得与三头六臂同等级别的击打能力，可以用于劈砍攻击。<br>● **额外手臂**：获得与三头六臂同等级别的力量属性或快速攻击能力(二选一)，落选的能力可以作为附带效果。<br>● **额外腿部**：可以移动得更快，获得与三头六臂同等级别的速度，其效果类似于跳跃能力。<br>● **尾巴**：可以像额外手臂那样使用尾巴，获得与三头六臂同等级别的快速攻击能力。<br>● **触手**：肩背或身体两侧长出强壮有力的触须，甚至是毛发状的触手卷须，获得与三头六臂同等级别力量属性或延展能力(二选一)，落选的能力可以作为附带效果。<br>● **翅膀**：长出能够飞翔的(鸟类、蝙蝠类或昆虫类)翅膀，获得与三头六臂同等级别的飞行能力。 |
| **拔地而起** | 你的体型增大，力量增强，但更容易暴露，也更容易被击中。你会获得“巨型”特质，启用这一特质即可获得优势或制造麻烦，此时你的力量等级等同于拔地而起的能力等级，或者等同于正常力量等级+1，两者取较大值，并获得与拔地而起同等级别的伤害抵抗能力。如果拥有2-5级拔地而起能力，你的防御检定和隐藏检定受到1点减值，6-8级时受到2点减值，9-10级时受到3点减值。胁迫体型较小的敌人并进行意志类机动动作检定时，效果则恰恰相反，拥有2-5级能力时获得1点加值，6-8级时获得2点加值，9-10级时获得3点加值(请参阅采取行动章节的机动动作部分)。 |
| **隐身** | 隐藏身形，连同穿戴携带之物一起不为人见，声音、气味、暴雨和其他类似途径仍然会出卖你的存在方位。如果被人察觉，你需要进行一次隐身检定，避免暴露踪迹。只要所在地点尚未暴露，敌人就无法直接向你发动攻击(不知道瞄准何方)，但爆溢效果等非直接攻击仍然会对你造成伤害。知晓你的位置所在时，敌人的攻击检定难度+2。 |
| **模仿** | 你可以复制或模仿其他角色/生物/物体的某些能力，需要一个画页的预先准备，而且每次只能模仿一个目标，更换目标即会失去已经模仿的能力。通过模仿获取的能力，其等级等同于模仿的等级或目标能力的等级(两者取较小值)。从以下选项之中选择一种进行模仿：<br><br>● **动物模仿**：你可以复制动物拥有的能力，比如大象的力量、猎豹的超高速移动、猴子的协调或鹰隼的远程视距和飞行等等。<br>● **物质模仿**：你可以复制接触到的物质或能量，其效果类似于变化形态(如前所述)，不同之处在于变化的形态取决于使用者复制的物质或能量。<br>● **植物模仿**：你可以复制植物拥有的能力，比如毒物和花粉引起的灾祸丛生、粗糙树皮带来的伤害抵抗、利用光合作用维系生命，通过不同花粉对昆虫施加精神控制等等。<br>● **特殊能力模仿**：你可以触碰目标，复制对方拥有的特殊能力，包括7级(或更高)的属性等级。 |
| **穿越固体** | 你可以摆脱实体形态——变成灵体、降低身体密度或以某种方式脱离物质世界等——免疫物理攻击，同时可以毫发无伤地穿越固体物质(以影响无实体目标作为附带效果的攻击/物质不包括在内)。如若穿越能量场(如力场)，需要进行一次穿越固体检定，难度等同于能量场的等级。处于无实体状态时，你无法在物理层面对世界造成影响，不过仍然可以使用精神系特殊能力(对抗物理目标难度+2)，同时也会受到这类能力的影响。如需增加附带效果，可以选择绝对生命维系(仅限无实体状态)。 |
| **微缩成寸** | 你可以随意缩小体型，每增加1个能力等级都可以使体型大约缩小到原来的一半。你会获得“小型”特质，启用这一特质即可获得优势或制造麻烦。对抗正常体型的敌人时，你的攻击检定和防御检定都会获得加值——1-4级能力获得1点加值，5-6级获得2点加值，7-8级获得3点加值。如果拥有9-10级微缩成寸，你的体型可以缩小到微观等级，甚至原子/亚原子等级，你将“超脱标准”，不再受到宏观世界的直接影响，只能与相同尺寸的事物互动，同时可以溜进极其微小的入口，甚至通过分子的间隙！ |
| **变形** | 你可以变成动物、物体或其他人的模样，从以下选项之中选择一种进行变形(其他选项可以作为附带效果)：<br><br>● **动物**：你可以变成普通动物或半人半兽的形态，保留原有的精神属性和语言能力(除非另有特殊限制)，身体属性等同于变形的能力等级或动物本身的属性等级(两者取较小值)。<br>● **人类**：你甚至可以模仿他人的声音和穿着，变化程度足以骗过指纹识别、视网膜验证和DNA检验，但能够模仿的只有外表，不包括能力(这部分内容请参阅特殊能力模仿)。<br>● **物体**：你可以变成岩石、墙壁或汽车等物体，保留原有的精神属性和语言能力(除非另有特殊限制/挑战)，同时获得目标的物理属性，如材质等级(最高不超过变形的能力等级)。 |
| **元素转化** | 你可以使触摸到的化学元素和化合物发生改变，由一种无生命物质转化为另外一种无生命物质，作用范围仅限于整个物体，可以作用的质量取决于元素转化的能力等级(参照基准表的重量纵列)。如果目标物体正在移动或被他人持有，你可能需要进行一次协调检定，否则无法触摸。如需增加附带效果，可以将能力的作用范围扩大到远程距离。 |

### 🎛️ 控制系 (Control) 特殊能力资料库

| 特殊能力名称 | 详细描述 (完全依照原文) |
| :--- | :--- |
| **变形射线** | 你可以在远程范围内，针对他人的密度体积、生长发育、可见程度、穿墙能力、体型大小和外观样貌等，选择一项予以改变。玩家需要进行一次协调检定，对抗目标的协调等级，成功即可命中目标，为他带来对应的改变效果。 |
| **宇宙力量** | 你可以连接来自宇宙的原始力量，复制一种特殊能力作为该项能力的附带效果。实际上，几乎所有特殊能力都可以作为宇宙力量的附带效果，从本质而言，宇宙力量的能力效果往往与能量控制非常类似(请参阅下文的能量控制)。 |
| **元素控制** | 你可以从空气、大地、火焰、植物、水源或气象之中选择一种元素予以支配，控制其走向和形态，其效果类似于同等级别的心灵遥感能力(参阅下文的心灵遥感)。如需增加附带效果，可以选择元素冲击波，也可以选择一些特殊效果——比如通过空气控制获得飞行能力，通过火焰控制获得灼热的光环，通过水源控制获得水下生存能力，或是通过大地控制获得挖掘洞穴能力等等。 |
| **能量控制** | 你可以从寒冰、黑暗、闪电、力量、重力、光线、磁性、辐射、声波和震动等能量之中选择一项予以控制，其效果基本上是对应能量的冲击波(光线控制或音波控制也可以达到眼花缭乱的效果)。如需增加附带效果，可以选择力场、飞行或心灵遥感等特殊能力，也可以选择光环、桎梏、眼花缭乱或击晕等其他攻击系特殊能力。 |
| **机关配件** | 通过制造千奇百怪的各式装置来获得形形色色的特殊能力。经过一个画页的预先准备，选择一项想要复制的特殊能力，然后进行一次智力检定(加上合适的专长加值)，对抗目标能力的等级——相当于难度值，最高不超过机关配件的能力等级。如果检定成功，你会得到能够复制目标能力的机关配件，持续时间为当前章节；如果检定失败，你需要一次痛下决心才能再次尝试。你也可以花费一个决意点，自动造出想要的机关配件，无须任何检定。如果失去所有装备，你的机关配件能力就会受到限制，或者完全无法使用，直到重新拿回装备。 |
| **治疗** | 可以恢复物理攻击造成的耐力损伤，需要接触目标，以一个行动动作恢复与治疗同等级别的耐力，最高不超过其耐力上限，且每期游戏为同一个目标恢复的耐力值不能超过使用者能力等级的二倍，每超出一倍的治疗量，就需要一次痛下决心。<br><br>治疗还可以使损失的力量等级得到恢复，这需要进行一次难度为2的治疗检定，如果检定失败，目标的力量等级得以恢复，但使用者会失去1个等级的力量，需要休息才能恢复！如果你的治疗不低于7级，那么无须检定即可自动成功。如需增加附带效果，可以不必考虑能力等级，取消所有治疗检定。 |
| **魔法** | 你可以施放法术，复制其他特殊能力的效果，需要以施放动作为限制条件(请参阅限制条件部分)，并且需要一个画页的预先准备，然后选择想要复制的特殊能力效果，进行一次难度等同于目标能力等级的检定(最高不超过魔法的能力等级)。选择魔法等级或一项属性数值固定用于施法检定，加上合适的专长加值(通常为神秘学专长)。如果施法检定成功，你能够获得对应等级的目标特殊能力；如果施法检定失败，你需要一次痛下决心才能再次尝试。你也可以选择花费一个决意点自动施放法术(无须检定)。如需增加附带效果，可以专精某一种特定法术，无须预先准备和施法检定即可施放与魔法同等级别的该项法术。 |
| **能力清除** | 你可以在远程范围内完全消除他人的特殊能力，需要进行一次协调检定，对抗目标的协调等级。如果检定成功，用目标各项能力的等级分别减去你的能力清除等级，结果为0(或更小)的特殊能力将失去作用，持续的画页数与能力清除的等级相同，其他特殊能力每画页恢复1个等级，直到回复正常等级。如若增加限定条件，可以让能力清除只对近身范围内的目标/特定类型的特殊能力生效。 |
| **概率控制** | 你可以对随机因素施以影响，获得与概率控制同等级别的额外决意等级(和正常决意点数一样能够恢复)，但只能用于启用能力附带的“运气”特质。确定能力时，从好运气和坏运气之间选择一项予以控制(或者掷骰决定：1-3为好运气，4-6为坏运气)。启用好运气可以获得加强尝试/回溯等对你有益的优势，启用坏运气可以制造增加难度/挑战等对人有害的麻烦。更多内容请参阅基本概念章节的优势部分和麻烦部分。如需增加优势，可以同时选择好运气和坏运气予以控制。 |

| **摇荡** | 你可以利用线绳或缆索——力量线束、自身生成的网状物或钩锚枪/套索之类的装置，可用的材质等级取决于摇荡的能力等级——使用摇荡/协调等级(两者取较大值)进行一次机动动作，还可以利用荡索抓住物体，进行一次协调检定，成功即可将目标拉拽到自己身边(而不会被拉拽过去)。如果想要将某物拽离某人身边，需要进行一次力量检定，对抗对方的力量等级。 |
| **瞬间移动** | 你可以从一个地点消失，然后出现在一段距离之外的另一个地点，无须穿过连接两个地点的空间，不同能力等级对应的移动距离请参照基准表。需要进行一次难度为2的瞬间移动检定，如果检定失败，你可以抵达目的地，但会陷入昏迷状态，需要将下个画格用于恢复(当前画页无法行动)。如果你的位面旅行不低于7级，那么无须检定即可自动成功。如需增加附带效果，可以不必考虑能力等级，取消所有瞬间移动检定。<br>● **传送**：如需增加附带效果，可以在两个地点之间建立双向传送门，维持传送门需要保持专注。 |
| **墙面爬行** | 你可以在垂直/颠倒的表面上(如墙面或天花板)移动自如。如果目标表面格外光洁平滑，游戏主持人可以要求使用者进行一次墙面爬行检定，难度值取决于其光滑程度。 |

### ⚔️ 攻击系 (Attack) 特殊能力资料库

| 特殊能力名称 | 详细描述 (完全依照原文) |
| :--- | :--- |
| **灾祸丛生** | 效果类似于疾病或毒药，玩家需要进行一次勇猛检定来触碰目标，然后进行一次灾祸丛生检定，对抗目标的力量/再生等级，两者取较大值。<br>● **一败涂地或重大失败**代表没有效果，灾祸丛生就此终止。<br>● **普通失败**代表当前画页没有效果，但灾祸丛生依然存在，玩家需要在下一画页属于自己的画格内再次进行检定。<br>● **成败之间**代表灾祸丛生对目标造成的耐力伤害只有能力等级的一半，无视伤害抵抗(但灾祸抵抗依然有效)，能力效果依然存在，玩家需要在下一画页属于自己的画格内再次进行检定。<br>● **普通成功或更好的结果**代表灾祸丛生对目标造成与能力等级相同的耐力伤害，无视伤害抵抗(但灾祸抵抗仍然有效)，能力效果依然存在，玩家需要在下一画页属于自己的画格内再次进行检定。<br>使用者可以选择灾祸丛生造成的效果，使目标的耐力降低到0并失去意识(等同一次重大成功的昏厥检定)，或者开始失去力量等级(等同于一次结果为重大成功的毙杀检定)。更多内容请参阅基本概念章节的伤害部分。 |
| **桎梏** | 通过投掷胶粘物、冰状物、泥浆或网织物等物质，对远程范围内的目标进行攻击，达到束缚或捕捉的目的，攻击物的材质等级等于桎梏的能力等级。使用者需要进行一次协调检定，对抗目标的协调等级：<br>● **失败**代表没有效果。<br>● **成败之间**代表角色对目标造成局部钳制(如下所述)，但攻击物的材质等级减半。<br>● **普通成功**代表角色对目标造成局部钳制，对手无法移动，所有行动受到-2减值。普通成功还可以使已经存在的局部钳制升级成为完全钳制。<br>● **重大成功或大获全胜**代表角色对目标造成完全钳制，对手彻底受到限制，无法进行任何身体行动，但逃脱不包括在内(请参阅采取行动章节的逃脱部分)。 |
| **冲击波** | 对远程范围内的目标发动一次具有伤害性的攻击，造成与冲击波同等级别的伤害值。你可以自行决定冲击波的细节描述——比如使用蛮力、元素还是能量——并决定这是一次冲击攻击还是一次射杀攻击。更多内容请参阅采取行动章节的攻击部分。如需增加附带效果，可以让冲击波既能发动冲击攻击也能发动射杀攻击，根据需求随时切换。 |
| **击打** | 你可以使用利爪、尖刺或类似的武器(剑、刀、锤等)施展近战攻击，需要进行一次勇猛检定，命中则会对目标造成与击打同等级别的伤害值。决定能力时，从钝打攻击和劈砍攻击之间选择一种作为击打的效果，前者造成的伤害至少为力量等级+1。如需增加附带效果，可以让击打既能发动钝打攻击也能发动劈砍攻击，根据需求随时切换。 |
| **光环** | 选择一种能够造成伤害的物质——火焰、原始能量、尖刺、强酸等——环绕在身体周围，所有接触到你的人都会受到与光环同等级别的伤害，包括徒手攻击者和持武器攻击者。故意触碰对手可以对其造成伤害，此时将光环造成的伤害视为普通力量伤害的叠加效果(请参阅标准附带效果部分的叠加伤害)。 |
| **眼花缭乱** | 通过炫彩夺目的光线、震耳欲聋的声响、化学喷雾、一块泥巴或其他类似手段，对目标的一种感官(获得能力时选定)造成影响的攻击方式，作用范围不仅包括正常的感官，也包括危险感知或察觉等感官系特殊能力，使用者需要进行一次协调检定，对抗目标(远程范围内)的协调等级：<br>● **失败或成败之间**代表没有效果。<br>● **普通成功**代表目标陷入眼花缭乱状态，持续时间为一个画页。<br>● **重大成功**代表目标陷入眼花缭乱状态，持续的画页数与能力等级相同。<br>● **大获全胜**代表目标陷入眼花缭乱状态，持续时间为整个章节。<br>如果角色陷入眼花缭乱状态，在进行与受扰感官相关的动作检定时会获得2点难度加值，进行与受扰感官相关的感知检定时会自动失败。使用优势进行恢复(请参阅基础规则章节的优势部分)即可立刻终止眼花缭乱状态。 |
| **能量吸收** | 可以通过接触使目标的生命力衰竭。为了能够触摸目标，你需要进行一次勇猛检定，如果检定成功，再进行一次能量吸收检定，对抗目标的力量/意志等级(选择能力时决定)，检定结果即为目标失去的耐力值，同时也是你获得的耐力值(如有耐力损失)，最多不超过能量吸收的能力等级。如需增加附带效果，可以允许使用者获得的耐力值超出能量吸收的能力等级，溢出的耐力值将每分钟流失1点。 |
| **快速攻击** | 你可以在一次行动时进行多次攻击，用快速攻击的等级除以攻击所用能力的等级，得到额外的攻击次数。如果在同一画页使用额外的攻击次数对抗同一名敌人，则按照齐心协力规则进行处理(请参阅基本概念章节的齐心协力部分)。 |
| **击晕** | 你可以致使远程范围内的敌人无法行动，需要进行一次协调检定，对抗目标的协调等级。如果攻击命中，再进行一次击晕检定，对抗目标的力量/意志等级(选择能力时决定)：<br>● **失败或成败之间**代表没有效果。<br>● **普通成功**代表目标失去行动能力，持续时间为一个画页。<br>● **重大成功**代表目标失去行动能力，持续的画页数与能力等级相同。<br>● **大获全胜**代表目标失去行动能力，持续时间为整个章节。<br>目标可以启用一种特质用于恢复(请参阅基本概念章节决意部分的恢复)，重新获得行动能力。你可以自行决定目标被击晕时的状态——眩晕、失去知觉、混乱或无力活动，抑或是因为疼痛、压力、恐惧或其他原因丧失行动能力。 |

### 👁️ 感官系 (Sensory) 特殊能力资料库

| 特殊能力名称 | 详细描述 (完全依照原文) |
| :--- | :--- |
| **察觉** | 你的感知等级等同于察觉的能力等级，可以发现视程范围内的特定之物——能量、特殊能力或精怪等。从天体、情感、能量、魔鬼、魔力、磁性、特殊能力、辐射或灵体之中选择(或自行设定)一项作为察觉的作用类型。如需增加附带效果，可以增加一种作用类型，或者将能力范围改为广距，让使用者的察觉和追踪能力不受距离所限。 |
| **超感知觉** | 你可以感知远处的事物，犹如身临其境，能够感知的距离请参照基准表。如若使用超感知觉留意/搜寻某种事物，你需要进行一次超感知觉/感知检定(两者取较小值)。如果目标地点能够屏蔽超感知觉，进行一次超感知觉检定，对抗目标地点的防护等级。如果目标地点是完全未知的领域，游戏主持人也可以要求使用者进行一次超感知觉检定，根据检定结果决定感知到的内容。 |
| **危险感知** | 一种危急关头的特殊“第六感”，可以用于抵御外界攻击或其他突如其来的危险(视为一个反应动作)，或是用于进行感知检定，察觉危险所在。如果你拥有合适的相关能力应对，其等级高于危险感知的能力等级，在检定时需要使用该项能力，并获得1点加值。 |
| **远程接入** | 你可以在视程范围内接入电脑并获取信息。操作电脑需要进行一次远程接入/智力检定(两者取较大值，加上合适的专长加值)，对抗电脑系统的智能程度(自我意识等)。远程接入的效果与心灵感应非常类似。 |
| **追溯感知** | 你可以感知过去发生的事情，需要接触目标地点/物品，并进行一次追溯感知检定，难度值取决于追溯年代的久远程度。<br>● **重大失败或更糟**的结果代表你可能会获得虚假信息或误导性的幻象，由游戏主持人视情况而定。<br>● **普通失败**代表你无法获得信息，需要一次痛下决心才能针对同一个目标再次尝试。<br>● **成败之间**代表你无法获得信息，但是可以再次尝试。<br>● **普通成功**代表你会获得含义模糊、可以有多种解释的线索和幻象。<br>● **重大成功**代表你会获得比较清晰的信息，可能是一个名字或一张面孔。<br>● **大获全胜**代表你会获得非常清晰详尽的幻象，不过未必所有信息都与探寻之事有关。<br>你还可以将追溯感知作为一个机动动作，利用感知到的内容获悉/制造特质并获取优势(请参阅采取行动章节的机动动作部分)。 |
| **预知未来** | 你可以看到未来可能发生之事的幻象，需要游戏主持人暗中进行一次预知未来检定，难度值取决于窥探之事的隐蔽/遥远程度。<br>● **重大失败或更糟**的结果代表你可能会获得虚假信息或误导性的幻象，由游戏主持人视情况而定。<br>● **普通失败**代表你无法获得信息，需要一次痛下决心才能针对同一个目标再次尝试。<br>● **成败之间**代表你无法获得信息，但是可以再次尝试。<br>● **普通成功**代表你会获得含义模糊、可以有多种解释的线索和幻象。<br>● **重大成功**代表你会获得比较清晰的信息，可能是一个名字或一张面孔。<br>● **大获全胜**代表你会获得非常清晰详尽的幻象，不过未必所有信息都与探寻之事有关。<br>你还可以将预知未来作为一个机动动作，利用预知到的内容获悉/制造特质并获取优势(请参阅采取行动章节的机动动作部分)。如需增加附带效果，可以选择危险感知。 |
| **超感官** | 你可以增加/增强/扩展感官能力，每增加1级能力，即可在以下增益之中选择一项：<br>● **增加感官**：每增加1个能力等级，你就可以额外拥有五感之外的一种感官能力，比如红外视觉、微观视觉、雷达视觉或X光视觉等。<br>● **增强感官**：每增加1个能力等级，你在进行特定感官的感知检定时就会获得1点加值(类似于专长)，比如增强视觉、增强听觉等。<br>● **扩展感官**：每增加1个能力等级，特定感官的作用距离就会扩展一个等级。如果你拥有1级扩展视觉，就可以在远程距离看到视程范围内的事物。 |
