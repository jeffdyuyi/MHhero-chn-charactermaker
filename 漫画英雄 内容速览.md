根据您提供的《漫画英雄》（遵循 2d6 核心机制的超级英雄 TRPG）规则书内容，我为您整理了**规则书核心章节大纲**，并为您制作车卡（角色创建）网页提取了**结构化的资料库内容**，方便您直接转化为 JSON/数据库 结构。

### 第一部分：《漫画英雄》核心章节内容整理

1. **基本概念 (Basic Concepts)**
   * **核心机制**：检定公式为 `尝试(能力等级+d6) - 难度(对抗能力等级+d6) = 结果`。
   * **等级与度量**：游戏中所有能力、属性、重量、速度等均被量化为 **1-10 级** 的统一标准（例如3级为人类平均，10级为登峰造极的超人类）。
   * **衍生属性**：**耐力**（承受伤害的血条）与 **决意**（用于重掷、发动特质、化险为夷的英雄点数）。
   * **伤害与恢复**：通过对比攻击与抵抗能力，造成减耐力、重击、昏厥甚至毙杀的效果。

2. **创建英雄 (Character Creation)**
   * 车卡包含六个主要阶段：**1.能力起源**（决定力量来源） -> **2.属性**（随机或购点决定6项基础数值） -> **3.特殊能力**（随机抽取超能力） -> **4.专长**（技能熟练度） -> **5.描述说明**（背景与外观） -> **6.特质**（角色的优缺点标签）。

3. **特殊能力 (Special Abilities) 与 装置 (Devices)**
   * 详细罗列了七大系（变形、控制、防御、精神、运动、攻击、感官）超能力的具体效果。
   * **附带效果（增强）与限制条件（削弱）**：允许玩家自定义能力的运作方式（如让火焰攻击附带控制效果，或限定某能力只能在夜晚使用）。
   * **装置**：包含护甲、武器、交通工具、机器人等，依赖实体物品的能力。

4. **专长 (Specialties)**
   * 角色的世俗技能（如武术、科学、驾驶等），分为基础(+1)、专家(+2)、大师(+3)三个加值级别。

5. **采取行动 (Taking Action)**
   * 规范了动作时间（战斗轮）、距离、移动（攀爬、跳跃、飞行等）、攻击行为（钝打、劈砍、冲击、射杀）以及环境互动（如陷阱、灾难）。

6. **游戏主持人 (Game Master) 与 NPC**
   * 指导主持人如何设计冒险（包括剧情框架、反派创建、提供灾难与死亡陷阱）。
   * 提供了常备角色模板（如路人、暴徒、军人、机器人）和常见动物/怪物的数值面板。

### 第二部分：车卡网页资料库构建准备 (Data Structure)

您可以将以下内容直接转化为前端状态树（如 Vue/React 的 State）或后端数据库表结构。

#### 1. 基础属性表 (Attributes)
数值范围限定为：**1 到 10**。
* `Brawn` (勇猛)：决定近战攻击与徒手搏斗。
* `Coordination` (协调)：决定敏捷、精准度、远程攻击、防御。
* `Strength` (力量)：决定肌肉力量、举重、耐力基础。
* `Intellect` (智力)：决定聪慧、理性思考、一般知识。
* `Awareness` (感知)：决定洞察力、察觉、细节关注。
* `Willpower` (意志)：决定人格力量、精神抗性、决意基础。

*(UI 提示：可做成拉条或数字输入框，并在旁边显示对应的评级，如：1-资质驽钝，3-平庸之辈，6-超乎常人，10-登峰造极)*

#### 2. 衍生属性公式 (Derived Stats)
* **耐力 (Stamina)** = `力量(Strength) + 意志(Willpower)`
* **初始决意点 (Resolve)** = `6 - 特殊能力总数量(Power Count)` （最低不能低于1）。

#### 3. 能力起源 (Origins)
玩家选择其一，会对初始建卡产生修正：
* **受训 (Trained)**：额外获得2项专长，可放弃1项超能力再换2项专长。
* **改造 (Altered)**：选择1项能力（属性或特殊能力）+2级（不超过10级）。
* **天赋异禀 (Mutant/Innate)**：额外获得1项特殊能力，或将1项能力+2级。
* **花招诡计 (Gimmick)**：特殊能力依赖装备，精神属性+2级。
* **人造生命 (Artificial)**：力量+2级，额外获得“维系生命”能力（等级等于舍弃的能力，或掷骰决定）。
* **天外来客 (Alien)**：选择2项能力各+2级。

#### 4. 专长字典 (Specialties)
**等级设定**：基础(+1加值) / 专家(+2加值) / 大师(+3加值且可放特技)
* **战斗类**：空中战斗、水下战斗、武器、武术、格斗。
* **驾驶类**：陆地驾驶、空中驾驶。
* **学识类**：科学、医学、神秘学、法律、精神病学、商业。
* **行动类**：运动技巧、调查、潜行、快手。
* **社交/其他**：艺术、领导力、语言、表演、工艺、精神抗性、军事、特殊能力。

#### 5. 特殊能力字典 (Super Powers)
*(车卡时可做成 7 个折叠面板，玩家从中挑选或随机掷骰)*

* **变形系 (Alteration)**：能力增幅、能力提升、释放自我、变化形态、水下生存、浓缩密度、基因复制、三头六臂、拔地而起、隐身、穿越固体、微缩成寸、动物模仿、物质模仿、植物模仿、能力模仿、伸缩自如、元素转化。
* **控制系 (Control)**：变形射线、元素控制、概率控制、时间控制、能量控制、治疗、心灵感应、变形、宇宙力量、机关配件、魔法、能力清除、仆役。
* **防御系 (Defense)**：凝神吸收、环境适应、力场、永生不朽、维系生命、反射攻击、再生、抵抗。
* **精神系 (Mental)**：灵体投射、梦境控制、情感控制、制造幻觉、生成幻象、精神冲击波、精神控制、精神护盾、心灵感应。
* **运动系 (Movement)**：挖掘洞穴、位面旅行、飞行、跳跃、高速旋转、超高速移动、摇荡、瞬间移动、墙面爬行。
* **攻击系 (Attack)**：灾祸丛生、桎梏、冲击波、击打、光环、眼花缭乱、能量吸收、快速攻击、击晕。
* **感官系 (Sensory)**：察觉、超感知觉、危险感知、远程接入、追溯感知、预知未来、超感官。

#### 6. 能力调整模组 (Modifiers)
*每项特殊能力都可以外挂以下 Tag：*
* **附带效果 (Extras, 增强能力)**：扩大范围、无视范围、撤销效果、叠加效果、减缓恢复。
* **限制条件 (Flaws, 削弱以换取减耗或强化)**：仅限动物、类型失效、消耗殆尽、缩小范围、保持专注、持续生效、逐级递减、排他性、仅限附带效果、仅限可视范围、仅限最高等级、特技禁用、特殊限定（如仅夜晚/仅对男性生效等）、仅限他人、施放动作、预先准备、限制来源、缩短时间、耐力消耗、不受控制、成败未知、变化莫测。

#### 7. 特质系统 (Qualities)
纯文本输入字段，用于描述角色的背景、性格、身份、弱点等（如“亿万富翁”、“怕火的火星人”、“我要复仇”）。
* **UI 建议**：提供 3~5 个 Text Input 框。特质不占用数值点数，在游戏中用于产生“优势”或“麻烦”以获取决意点。
